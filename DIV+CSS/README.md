## CSS模块化
CSS3语言开发朝着模块化发展的。作为一个模块，以前的规范太庞大，而且比较复杂，所以，把它分解为一些下小的模块，使更多新的模块被加进来。这些模块包括:盒子模型、列表模块、超链接方式、语言模块、背景和边框、文字特效、多栏布局等。
